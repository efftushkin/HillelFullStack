About This Project

Project Information

This website is Homework #3 for the Fullstack JS Course.

Project Details

Assignment:
Homework #3
Course:
Fullstack JS
Topic:
HTML Structure and Navigation
Focus:
Creating a multi-page website with proper navigation, forms, and semantic HTML

Learning Objectives

This project demonstrates the following skills and concepts:

  • Creating a proper file structure for a multi-page website
  • Implementing consistent navigation across multiple pages
  • Using semantic HTML5 elements
  • Building complex forms with various input types
  • Creating image galleries with links
  • Implementing anchor links for page navigation
  • Organizing content in a logical and accessible manner

Technologies Used

  • HTML5
  • Semantic markup
  • Forms and input validation
  • Navigation and anchor links

Content Theme

The website showcases the Samorost game series by Amanita Design, a Czech independent game development studio. The series is known for its unique artistic style, atmospheric music, and engaging point-and-click adventure gameplay.
