Вивчення CSS властивостей

1. Display: block

Блокові елементи займають всю доступну ширину та починаються з нового рядка.

2. Display: inline

Інлайнові елементи займають тільки необхідну ширину і не переносяться на новий рядок.

6. Display: none

Елемент повністю видаляється з потоку документа і не займає місця.

7. Box-sizing: content-box (за замовчуванням)

Width та height застосовуються тільки до вмісту. Padding та border додаються зверху.
Загальна ширина = width + padding + border = 200px + 40px + 10px = 250px

Width: 200px
Padding: 20px
Border: 5px
Загальна ширина: 250px

8. Box-sizing: border-box

Width та height включають padding та border.
Загальна ширина = width = 200px (padding та border включені)

Width: 200px
Padding: 20px
Border: 5px
Загальна ширина: 200px

Висновки

  • display: block - елемент займає всю ширину
  • display: inline - елемент в потоці тексту
  • display: inline-block - в потоці, але з можливістю задати розміри
  • display: flex - гнучкий одновимірний макет
  • display: grid - двовимірна сітка
  • display: none - елемент прихований
  • box-sizing: content-box - розміри без padding/border
  • box-sizing: border-box - розміри включають padding/border (зручніше!)
